|----|--------|------|------------------|-------------|

## Trait Profile

τ = [τ₁, τ₂, τ₃, τ₄, τ₅, τ₆, τ₇, τ₈, τ₉] where τᵢ ∈ [0, 1]

## Cognitive State

C_net = Σ(τᵢ * vᵢ)

The cognitive state is a weighted sum of basis vectors.
